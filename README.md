# Movie
2019 '스마트모바일 프로그래밍' 개인 프로젝트
>- "나만의 작은 영화 기록소"
>- 영화에 대한 별점과 감상평을 다양하게 기록할 수 있는 애플리케이션입니다.
>- 리스트뷰, 인텐트, 암시적 인텐트, 웹뷰, 갤러리, 데이트피커, 파일 입출력, 캔버스...

1. 개발자: 조희준\
   개발 기간: 2019/6/21
   
2. 기능
- 메인 화면에 영화 추가
- 영화 사이트와 각종 url을 통한 영화 검색
- 신작 추천
- 영화에 대한 별점과 감상평 저장, 수정
- 영화에 대한 그림 일기 저장, 수정, 삭제
